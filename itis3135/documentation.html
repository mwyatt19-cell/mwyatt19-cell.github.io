Step One

Welcome to making your first ocean themed web page! We are going to be using a mix of HTML and CSS. First, we will make a basic webpage to add designs onto.

  1. Open an IDE such as Visual Studio Code
  2. Create a new HTML file
  3. Acess the boilerplate using the ! Emmet Abbrieviation and hit enter

Now that you have the base frame for your website, use a loren ipsum generator to add some text to your webpage. This will give you a better idea of how the design will look with text on it.

Step Two

Now that you have the basic webpage set up, we can start adding some design elements to make it look more ocean themed. We will be using CSS to add colors, fonts, and images to our webpage.

First, lets change the background color of our webpage to represent the ocean. We can add in a generic color, like blue, add the hex code of a color, or use hsl(hue, saturation, lightness) to get the exact color you want. 

        
          *
          {
            background-color: blue;
          }
        
        
        
           *
          {
            background-color: #00b4d8;
          }
        
      
         
           *
          {
            background-color: hsl(240, 80%,  80%);
          }

Step Three

Now that we have ocean-like background-color, lets change the font color to match our theme, while agreeing with the WCAG guidelines for accessibility. We can use the same color methods as before to change the font color. We can use a program like Coolors and use the Contrast Checker to find colors that match our theme and are accessible.

Remember, the minimum contrast ratio for normal text is 4.5:1, and for large text is 3:1. Large text is defined as 18pt and above, or 14pt and above if it is bold. 

        
          *
          {
            background-color:#00B4D8;
            color:##000732;
          }

Step Four

We have the background color and font color. Now, what about the actual font of our text? Using a program such as Google Fonts, we can find free fonts to use on our webpage. We can search for fonts that match our theme, and then add the link to the font in our HTML file, and then use the font-family property in CSS to apply the font to our text.

For your heading, using a Serif font, and for the rest of the text, using a Sans Serif font to create a nice contrast between the two. Don't forget to import the fonts into your HTMl Code as well! 

        
          header
            {
              font-family: "Maiden Orange", serif;
            }
    
          *
          {
            font-family: "Keania One", sans-serif;
          }

Step Five

Now lets tackle some formatting to ensure our code stays readable for later. Below is a list of guidelines and steps to check on your code.

  • Does the name of your HTML File make sense?
  • Does the name of your CSS File make sense?
  • Has the CSS file been linked to your HTML Document?
  • Does your code meet WCAG guidelines?
  • Is your code formatted/indented correctly (tools such as Prettier are helpful to ensure correct formatting)?

Now you have a basic ocean themed webpage! You can continue to add more design elements, such as images, animations, and more to make it your own. Don't forget to test your webpage on different devices and browsers to ensure it looks good and functions properly for all users.
